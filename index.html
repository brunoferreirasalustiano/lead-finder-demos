Dúvidas frequentes

O essencial antes de começar.

O escopo básico é objetivo para manter prazo, qualidade e preço de lançamento.

O site funciona no celular?

Sim. A experiência é criada primeiro para telas móveis e adaptada para tablets e computadores.

O domínio está incluído?

O desenvolvimento e a publicação inicial estão incluídos. Um domínio próprio, quando desejado, é contratado separadamente.

A página recebe pagamentos ou agenda horários?

O plano essencial direciona o cliente para o WhatsApp. Loja virtual, agenda integrada e pagamentos dentro do site são recursos adicionais.

O site garante primeira posição no Google?

Não. A página recebe uma base técnica de SEO, mas posicionamento depende de concorrência, conteúdo, reputação e trabalho contínuo.
