# Lead Finder Brasil — Demos

Catálogo estático e responsivo de landing pages demonstrativas para pequenos negócios.

## Segmentos

- barbearia;
- oficina mecânica;
- restaurante;
- prestadores de serviços.

## Oferta apresentada

Landing Page Essencial a partir de R$ 650, com Pix ou cartão em até 10x conforme as condições da operadora.

## Privacidade

As empresas, endereços, valores e informações das demonstrações são fictícios. O site não contém formulários, analytics, tracking ou envio real para WhatsApp. O número comercial permanece desativado até configuração explícita.

## Validação

```bash
node scripts/validate-static-site.mjs
```

## Publicação

O GitHub Pages é validado e publicado automaticamente após alterações aprovadas na branch `main`.
