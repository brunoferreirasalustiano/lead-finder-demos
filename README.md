# Lead Finder Brasil — Landing Pages Demonstrativas

Catálogo estático e responsivo com quatro demonstrações comerciais modernas:

- Barbearia e salão de beleza
- Oficina mecânica e auto center
- Restaurante e lanchonete
- Prestadores de serviços locais

## Objetivo

Apoiar o piloto interno da Lead Finder Brasil com experiências visuais que possam ser apresentadas somente depois da autorização do potencial cliente.

## Direção visual

O projeto utiliza um design system autoral baseado em HTML semântico, CSS moderno e JavaScript enxuto. A experiência inclui:

- tipografia grande e hierarquia editorial;
- composição assimétrica e bento grids;
- mockups construídos somente com HTML e CSS;
- formas orgânicas, gradientes e camadas translúcidas;
- temas próprios para cada segmento;
- animações de entrada, contadores, FAQs e tabs acessíveis;
- navegação responsiva e suporte a redução de movimento.

## Princípios

- nenhuma empresa real é representada;
- nomes, preços e endereços são fictícios;
- sem rastreamento, cookies ou coleta de dados;
- sem integração real com WhatsApp;
- botões comerciais são apenas demonstrativos;
- sem fontes, imagens, scripts ou bibliotecas externas;
- publicação somente após revisão e aprovação.

## Execução local

```bash
python -m http.server 8080
```

Depois acesse `http://localhost:8080`.

## Publicação no GitHub Pages

O projeto foi preparado para publicação direta a partir da branch `main`, na raiz do repositório. Em **Settings → Pages**, selecione **Deploy from a branch**, escolha `main` e `/ (root)`.
