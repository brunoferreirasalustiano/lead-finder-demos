Privacidade

Um site demonstrativo sem coleta direta de dados pessoais.

Esta página explica, em linguagem direta, quais recursos o site utiliza e o que acontece quando o visitante escolhe entrar em contato.

Resumo

O site não possui mecanismos próprios de rastreamento.

0

Formulários

Nenhum nome, telefone, e-mail ou mensagem é digitado ou enviado dentro deste site.

0

Cookies próprios

O catálogo não cria cookies para publicidade, perfil de usuário ou acompanhamento de navegação.

0

Analytics

Não há Google Analytics, pixel de publicidade, Hotjar ou ferramenta semelhante instalada.

0

Armazenamento

O site não possui banco de dados, autenticação, conta de usuário ou armazenamento local.

WhatsApp

O contato acontece fora deste site.

O que o botão faz

  • abre o endereço oficial `wa.me`;
  • preenche uma mensagem inicial;
  • deixa o visitante decidir se deseja enviar;
  • funciona sem envio automático pelo site.

O que o site não faz

  • não acessa conversas;
  • não confirma que a mensagem foi enviada;
  • não utiliza WhatsApp Cloud API;
  • não automatiza WhatsApp Web ou disparos em massa.
Serviços externos

Recursos carregados por terceiros.

A hospedagem é realizada pelo GitHub Pages. Algumas imagens demonstrativas ainda são carregadas do Unsplash. Ao abrir o WhatsApp, o visitante passa a utilizar um serviço externo sujeito às políticas da respectiva plataforma.

Esses serviços podem registrar dados técnicos de conexão conforme suas próprias políticas. A Lead Finder Brasil não recebe esses registros por meio deste site e não instalou ferramentas próprias para acompanhá-los.
